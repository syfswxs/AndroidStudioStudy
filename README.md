## 使用AndroidStudio的一些小技巧
### 代码自动排版快捷键
  * Ctrl + Alt + L
### 自动导入包快捷键
  * Alt + Enter
### 展开收缩代码快捷键
  * 使用Ctrl Shift +或-，就可以展开或收起全部代码。
  * Ctrl + 或 - 对当前方法展开或者收起
### Toast提示框
  * 代码：
  ```java
  Toast.makeText(JgjlActivity.this,"显示的文本:",Toast.LENGTH_SHORT).show();
  ```
    其中JgjlActivity为当前activity的名称，根据实际情况更改
    提示框内容也根据需要更改
